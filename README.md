# 🌿 Velvia - Tienda Online de Productos Artesanales

Proyecto Intermodular de Desarrollo de Aplicaciones Web enfocado en la creación de una tienda online completa con funcionalidades reales de ecommerce.

## Tecnologías utilizadas

- PHP (backend)
- MySQL (base de datos)
- HTML5
- CSS3
- JavaScript
- XAMPP (entorno local)

---

## Funcionalidades principales

### 👤 Usuarios
- Registro e inicio de sesión
- Gestión de perfil
- Direcciones de envío

### 🛍️ Catálogo
- Productos dinámicos desde base de datos
- Categorías (velas, bálsamos, jabones, etc.)
- Packs de productos

### 🛒 Carrito
- Añadir productos
- Eliminar productos
- Contador de productos en el header

### 💳 Checkout
- Selección de dirección
- Generación de pedido

### 📦 Pedidos
- Historial de pedidos del usuario

### Panel de administración
- Acceso restringido por rol (admin)
- CRUD completo de productos:
  - Crear
  - Leer
  - Editar
  - Eliminar
